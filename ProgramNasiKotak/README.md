# Program Pemesanan Nasi Kotak Restoran "Terserah"

Program Pemesanan Nasi Kotak Restoran "Terserah" ini dibuat untuk mengitung dan menyimpan data tagihan pemesanan nasi kotak di Restoran "Terserah" sesuai dengan cabang yang dipilih. Program ini dibuat menggunakan bahasa pemrograman **HTML (Hypertext Markup Language)**, **CSS (Cascading Style Sheet)** dan **PHP (Hypertext Preprocessor)** serta library **Bootstrap**.

## Requirements

* XAMPP : PHP >= 8.0.0
* Google Chrome >= 89.0.4389.114

## Instalation

* XAMPP

   Download [XAMPP](https://www.apachefriends.org/download.html) sesuai OS (Operating System) masing-masing, kemudian install sesuai petunjuk.
   
## Usage

1. Letakan folder **ProgramNasiKotak** ke dalam 
    > *C:\xampp\htdocs*  (Windows).
2. Jalankan *XAMPP -> Apache*.
3. Buka Browser ketikan 
   > *localhost/ProgramNasiKotak/*
